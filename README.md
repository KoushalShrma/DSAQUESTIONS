# DSAQUESTIONS

## Overview
This repository contains a comprehensive collection of 153 Data Structures and Algorithms questions with C++ solutions.

## Document Structure
The main document `DSA_Questions_Collection.docx` contains:

### Question Distribution
- **Easy Questions**: 77 problems
- **Medium Questions**: 54 problems  
- **Hard Questions**: 22 problems
- **Total**: 153 problems

### Features
- All solutions are written in C++ with LeetCode function signatures
- Questions are sourced from LeetCode platform
- Solutions include both brute force and optimized approaches
- No comments in code solutions for clean presentation
- Proper categorization by difficulty level

### Topics Covered
- Array manipulation and searching
- String processing and pattern matching
- Linked list operations
- Binary tree traversal and operations
- Dynamic programming
- Backtracking algorithms
- Graph algorithms
- Sorting and searching
- Mathematical problems
- Bit manipulation

### File Contents
- `DSA_Questions_Collection.docx` - Main Word document with all 153 questions
- `generate_dsa_questions.py` - Python script used to generate the document
- `README.md` - This documentation file

### Usage
Open the Word document to access all DSA questions with their descriptions and C++ solutions. Each question includes:
- Problem title and description
- LeetCode URL reference
- Complete C++ solution with proper function signature

### Solution Types
The solutions provided include:
- Brute force approaches for learning
- Optimized solutions for interviews
- Multiple solution strategies where applicable

Perfect for:
- Interview preparation
- Coding practice
- Algorithm learning
- DSA concept reinforcement
